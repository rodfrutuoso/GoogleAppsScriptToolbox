# Descrição
 O "Google Apps Script Toolbox" é um repositório aberto que reúne uma coleção de scripts desenvolvidos para aprimorar a funcionalidade e a automação do Google Workspace, com foco no Google Sheets.

 # Description
 The "Google Apps Script Toolbox" is an open repository that brings together a collection of scripts designed to improve the functionality and automation of Google Workspace, especially Google Sheets.
